Technology Stack

Chosen for rural-Bharat constraints

Low-bandwidth resilience, vernacular NLP, and rapid iteration for a 5-person founding team.

Mobile
Flutter (Dart)
Single codebase, smaller APK than React Native for low-end Android.
AI Layer
Claude API + intent classifier
Vernacular reasoning with structured triage output via tool use.
Speech
Bhashini + Whisper fallback
Native multilingual ASR/TTS across 10+ Indian languages.
Video
WebRTC (Jitsi self-hosted)
Low-cost, low-bandwidth, no per-minute licensing.
Database
PostgreSQL (Supabase)
Relational integrity for health records, row-level security.
Hosting
AWS ap-south-1 + ECS Fargate
India data residency, stateless scaling.
Compliance
ABDM + FHIR + DPDP 2023
Interoperable with govt. health stack, consent-aligned.
Safety
Deterministic rules-engine
Overrides AI on red-flag symptoms, routes to emergency care.
